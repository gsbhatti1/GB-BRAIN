> Name

Hedge 2.7 Rate of Return Statistics System - Supports Multiple Account Statistics

> Author

Zero

> Strategy Description

The hedge 2.7 rate of return statistics system supports multiple groups of account statistics. You can start statistics by adding exchanges with an even length. Every two are added as a group.
The currency price will be based on the first exchange’s bid price as a reference value.

> Strategy Arguments



|Argument|Default|Description|
|----|----|----|
|Interval|60|Statistical interval (seconds)|
|InitMode|0|Mode for counting initial funds: automatic statistics|manual input|
|InitBalance|false|Initial total money|
|InitStocks|false|Initial total coins|
