> Name

Single-point Snipe High-frequency Replenishment Auto-Hand reversal Algorithm-V12

> Author

> Strategy Description

Open a position first. Close the position at a specified profit point, if unable to close, exceed the stop-loss point and then replenish to reduce the average entry price. Continue replenishing until a profitable exit is achieved. If replenishment reaches the limit of funds or assets, start doing the opposite direction trade; previously short-selling, now go long-buying. This process continues in a loop.

Alternatively, you can disable the auto-reversal and let the program wait for unwinding.

For example, if currently holding an average 10 yuan long position with a target profit of 5 jiao, set up an order to sell at 10.5 yuan. If the price falls, replenish near the current price to adjust the average entry price and place another target-profit order. If unsuccessful, continue to replenish to lower the average entry price; if unable to recover, get trapped in a loss, then go short-sell.

The auto-reversal mechanism of the strategy

If currently holding a long position with profit, it will automatically open more long positions; if trapped, switch to opening short positions. After profitable short selling, continue short-selling until replenishment reaches its limit and start opening long positions again. This cycle continues as long as the previous trade was profitable.
The strategy calculates the necessary replenishment quantity and new target-profit values.

Noted:

This strategy has a complete recovery mechanism that can be used for practical operation or learning purposes.
To ensure 100% profitability, you must have sufficient funds to replenish. 
If insufficient funds are available, enabling auto-reversal will result in floating profits and losses.
With adequate capital, you can continue automatic replenishment without needing auto-reversal.

The strategy parameters can be adjusted to implement a high-frequency market-making algorithm.
It is not suitable for futures trading as it may lead to margin calls; therefore, it should only be used with spot trading.
