> Strategy Description

Set an investment amount, such as 10,000 yuan. Set a proportion between funds and Bitcoin, for example, 50:50. Maintain the ratio of funds to the value of Bitcoin based on current prices.

For instance, with initial capital of 10,000 yuan, set the fund-to-Bitcoin ratio at 30:70. In the first trading session, buy 7,000 yuan worth of Bitcoin (assuming a coin price of 1,000 yuan, purchasing 7 coins), leaving 3,000 yuan as remaining funds. The ratio should be maintained at 30:70.

In the next trading period, if the coin price changes to 1,500 yuan, the total value becomes 1,500 * 7 + 3,000 = 13,500. To maintain the proportion of funds and Bitcoin at 30:70, the amount of funds should be 13,500 * 0.3 = 4,050 yuan. Therefore, sell 4,050 - 3,000 = 1,050 yuan worth of Bitcoin, which is 1,050 / 1,500 = 0.7 coins. The remaining Bitcoin would be 7 - 0.7 = 6.3 coins, with a value of 6.3 * 1,500 = 9,450 yuan. The ratio of funds to the value of Bitcoin is now 4,050:9,450 = 30:70.

Similarly, when the coin price falls, buy more to maintain the proportion unchanged.

> Strategy Arguments


|Argument|Default|Description|

|UseAccount|true|Use all funds in the account|

|StockRatio|0.418|Coin value proportion to total value|

|ErrorInterval|300|Retry interval on error.|

function CaculateHoldings() {

    var ratio = 0;

    stockValue = stockHold * currentPrice;
    totalValue = stockValue + moneyHold;

    ratio = stockValue / totalValue;
    Log('Current Price:', currentPrice);
    Log('stockHold:', stockHold);
    Log('moneyHold:', moneyHold);
    Log('stockValue:', stockValue);
    Log('Current Ratio:', ratio);

    if (ratio > StockRatio) { //Sell
        Log('Current ratio > StockRatio');
        var sellAmt = (stockValue - totalValue * StockRatio) / (currentPrice - SlidePrice);
        lastOrderID = exchange.Sell(currentPrice - SlidePrice, sellAmt);
    } else { //Buy
        Log('Current ratio < StockRatio');
        var buyAmt = (totalValue * StockRatio - stockValue) / (currentPrice + SlidePrice);
        lastOrderID = exchange.Buy(currentPrice + SlidePrice, buyAmt);
    }
    Log('Order ID:', lastOrderID);
    CaculateHoldings();

}
