The algorithm uses rolling volatility based on log price changes over a certain period to calculate the highest and lowest values within a given interval. The upper channel is set by the maximum value and the lower channel by the minimum value, with entry when breaking above the upper channel. Both channels’ moving averages serve as exit lines. (Pay close attention here!)
